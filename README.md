**Windows**

**macOS:**

**Linux:**
  1. Unduh file .deb atau .rpm yang sesuai dengan distribusi Linux kalian.
  2. Buka terminal dan jalankan perintah berikut:
     ```bash
     sudo dpkg -i TownVille.deb
     ```
     atau
     ```bash
     sudo rpm -i TownVille.rpm
     ```
